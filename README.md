# BEE2041 Empirical Project: Macroeconomic Indicators and the Stock Market

## Project Overview

The project is formatted as a data-driven blog post that explores the structural relationship between macroeconomic indicators (such as GDP growth, unemployment, inflation, and interest rates) and the stock market between the volatile period of 2000 and 2008. The analysis is conducted through the lens of historical materialism, examining cycles of accumulation, the centralisation of capital, and the labor market.

## Repository Structure
- `blog.ipynb`: The main Jupyter Notebook containing the data-driven blog post, including the narrative text and all Python source code for data cleaning, exploratory data analysis, and Unit 5 advanced modeling (OLS Regression).
- `finance_economics_dataset.csv`: The macroeconomic dataset used for the analysis, obtained from Kaggle.
- `requirements.txt`: A list of required Python packages to run the notebook.
- `README.md`: This documentation file.

## How to Replicate

To replicate this work and view the analysis from scratch, follow these steps:

1. **Clone the repository:**
   Download or clone this repository to your local machine to ensure you have both the notebook and the dataset in the same directory.

2. **Install dependencies:**
   Ensure you have Python installed. It is highly recommended to use a virtual environment. Install the required packages using the provided `requirements.txt` file by running the following command in your terminal:
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch Jupyter:**
   Start the Jupyter Notebook server from your terminal in the project directory:
   ```bash
   jupyter notebook
   ```
   This will open a browser window. Click on `blog.ipynb` to open the project.

4. **Execute the Code:**
   To replicate the analysis, run all cells from top to bottom (In the top menu, select `Kernel` > `Restart & Run All`). This will execute the data cleaning steps, generate the 6 visualizations, and output the regression model summary.

## Output
The formatted output of this project is contained entirely within the `blog.ipynb` file.
